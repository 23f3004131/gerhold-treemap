# Gerhold Inc - Market Share Analysis Treemap

**Email:** 23f3004131@ds.study.iitm.ac.in

## Project Overview
This repository contains a professional Treemap visualization created for Gerhold Inc's strategic consulting project. The visualization analyzes market share distribution across product categories and subcategories for a Fortune 500 client's portfolio analysis and resource allocation decisions.

## Visualization Details
- **Chart Type:** Treemap
- **Tool Used:** RAWGraphs (https://rawgraphs.io/)
- **Data Structure:** Hierarchical data with category, subcategory, and market value
- **Total Data Points:** 20 rows across 6 major categories
- **Format:** PNG (300-512px dimensions)

## Data Categories
1. **Electronics** - Smartphones, Laptops, Tablets, Wearables, Audio Devices
2. **Clothing** - Women's Apparel, Men's Apparel, Children's Wear, Accessories
3. **Home & Living** - Furniture, Kitchen Appliances, Home Decor
4. **Healthcare** - Medical Devices, Personal Care, Wellness Products
5. **Food & Beverage** - Packaged Foods, Beverages, Snacks
6. **Automotive** - Parts & Accessories, Vehicle Electronics

## Files
- `README.md` - Project documentation (this file)
- `chart.png` - Treemap visualization generated using RAWGraphs
- `data.csv` - Source data used for the visualization

## Business Context
This visualization supports executive decision-making by providing an intuitive view of market share distribution, making it easy to identify dominant product categories and subcategories at a glance. The hierarchical structure allows stakeholders to understand both high-level portfolio composition and detailed subcategory performance.
