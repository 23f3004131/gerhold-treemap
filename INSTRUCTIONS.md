# RAWGraphs Treemap Creation Instructions

Follow these steps to create your professional Treemap visualization:

## Step 1: Copy the Data
Open the `data.csv` file in this folder and copy all the contents (including the header row).

## Step 2: Visit RAWGraphs
Go to https://rawgraphs.io/ in your web browser.

## Step 3: Import Your Data
1. On the RAWGraphs homepage, scroll to the "Load your data" section
2. Paste your CSV data into the text area
3. RAWGraphs will automatically detect the structure
4. You should see a preview of your data with 20 rows

## Step 4: Select Treemap Chart
1. Scroll down to the "Choose a chart" section
2. Look for the **Treemap** chart type
3. Click on it to select it

## Step 5: Map Your Dimensions
In the "Map your dimensions" section:
1. Drag **category** to the **Hierarchy** field (first level)
2. Drag **subcategory** to the **Hierarchy** field (second level, below category)
3. Drag **value** to the **Size** field
4. Optionally, drag **category** to the **Color** field for color coding by category

## Step 6: Customize Your Chart
In the "Customize your chart" section:
1. **Colors**: Choose a professional color scheme (e.g., "Category10" or "Tableau10")
2. **Padding**: Set to 2-5 for clear separation between boxes
3. **Labels**: Enable labels for both categories and subcategories
4. **Show hierarchy**: Enable to show category boundaries
5. Adjust font sizes for readability (10-12pt recommended)

## Step 7: Export Your Visualization
1. Scroll to the "Export" section at the bottom
2. Select **PNG** format
3. Set dimensions to **512x512 pixels** (or between 300-512px)
4. Click **Download** button
5. Save the file as `chart.png` in this folder

## Tips for Professional Results
- Use colors that distinguish categories clearly
- Ensure labels are readable (white text on dark boxes or vice versa)
- Make sure the largest boxes (highest values) are easily identifiable
- Preview the chart before exporting to ensure quality

## Verification Checklist
✓ Chart shows hierarchical structure (categories containing subcategories)
✓ Box sizes reflect the market values correctly
✓ Labels are clear and readable
✓ Professional color scheme applied
✓ Image is 300-512px in dimensions
✓ File saved as chart.png in PNG format

Once you've completed these steps, your visualization will be ready for GitHub upload!
